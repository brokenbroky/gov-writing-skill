# 🧠 Gov Writing AI V3（体制内公文写作AI系统）

## 🎯 项目简介

这是一个AI驱动的体制内公文写作系统，可以将任何零散信息自动转换为标准机关材料，包括：

- 汇报材料
- 工作总结
- 讲话稿
- 调研报告

---

## 🚀 核心能力

### 1. 输入即结构化
支持：
- 口语描述
- 关键词
- 混乱笔记
- 数据内容

---

### 2. 自动结构生成
自动生成：
- 基本情况
- 工作举措
- 存在问题
- 下一步打算

---

### 3. 机关语言转换
将口语转为公文表达：
- “很多问题” → “问题仍较为突出”
- “推进慢” → “节奏有待加快”
- “还可以” → “取得阶段性成效”

---

### 4. 成稿输出
支持三种模式：
- 标准上报版
- 领导汇报版
- 讲话扩展版

---

## 🧠 系统本质

> 将“非结构化信息”转化为“机关标准表达系统”

---

## 📦 使用方式

将 system_prompt.txt 作为 AI system prompt 使用即可。
